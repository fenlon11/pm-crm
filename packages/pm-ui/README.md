# Twenty UI

This library was generated with [Nx](https://nx.dev).

## Building

Run `npx nx build pm-ui` to build the library.

## Storybook Server

Run `npx nx start pm-ui` to start the storybook development server on `localhost:6006`.

## Running unit tests

Run `npx nx test pm-ui` to execute the unit tests via [Jest](https://jestjs.io).
